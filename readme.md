本项目面向无先验地图条件下轮足机器人的长距离自主导航，研究对象为搭载三维激光雷达、IMU和边缘计算平台的Unitree Go2W机器人。系统利用FAST-LIO2提供机器人位姿估计与点云运动补偿，在机器人周围实时构建包含地面高度、顶部障碍高度和垂直净空信息的局部环境表示，用于描述台阶、坡道、坑洼、粗糙地面、普通障碍及低矮通道等复杂场景。在此基础上，研究面向远距离目标的局部目标生成方法，根据当前位姿、目标方向、已观测环境和历史导航状态，持续生成具有可达性和方向引导作用的局部目标点，具体生成机制将在后续研究中进一步确定。局部规划器结合局部环境、局部目标和历史运动信息，输出局部路径、短时轨迹或速度指令，并由Go2W原有低层控制器完成特殊地形通过。通过局部目标生成、地形感知局部规划和滚动执行，实现未知环境中的长距离连续导航。目前已经具备了定位功能，下一步是局部环境表示模块。
本项目采用三维激光雷达和IMU获取机器人周围环境信息，利用FAST-LIO2完成点云运动补偿、位姿估计和多帧点云对齐。在此基础上，复用ROG-Map构建机器人中心的局部三维占据地图，并通过自定义投影模块提取地面高度、顶部障碍高度及二者之间的垂直净空信息，形成双层高程—净空地图。该地图能够统一表示平地、台阶、坡道、坑洼、墙体、立柱和低矮通道等环境结构，同时区分已观测自由空间与未知区域。最终采用Grid Map组织并发布多层局部环境信息，为后续机器人能力相关的可通行性判断、局部目标生成和学习式局部规划提供统一输入。整体流程为：XT16与IMU输入FAST-LIO2，注册点云与位姿输入ROG-Map，局部三维占据地图经过双层高度投影，生成地面高度、顶部高度、净空及阻塞信息，最终输出机器人中心局部环境地图。
